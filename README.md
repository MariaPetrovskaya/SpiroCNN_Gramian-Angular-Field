# SpiroCNN_Gramian-Angular-Field
Light neural network for medicine task from scratch. (Proof of Concept stage, May 2025).
The task was to create model and to train the model to recognize and classify spirometry images based on given features.
The features were in the csv dataset and in separate .png images files (spirometry graphs).
The target was "M_Acceptable" column in dataset.
The notebook contains several approaches and runs of various models.
